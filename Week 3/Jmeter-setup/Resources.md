# JMeter Resources

## Firefox
It is not the only browser you can use for recording your performance test scripts, but it is the easiest to use when setting up test script recording on your local computer. Firefox can be downloaded [here](https://www.firefox.com/en-US/?redirect_source=mozilla-org).

## Java Runtime Environment
JMeter is a Java application, so it needs a JRE to run. Since we are also going to be running the Todo application locally we want to aim for JRE 21 at a minimum. You can use your editors like VSCode and Intellij to download the JDK you need (which contains the JRE as well) or you can go to a distributor's website and download the environment directly. The distribution used in training is the Eclipse Temurin distribution found [here](https://adoptium.net/temurin/releases/).

## JMeter
In order to run JMeter we actually need JMeter. JMeter can be downloaded in a zip file from the JMeter website and then set up on your local machine with minimum configuration needed. You can find the download details for JMeter [here](https://jmeter.apache.org/download_jmeter.cgi).
